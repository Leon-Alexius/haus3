Haus 3 Board is a small Astro site for news, updates, and feedback intake.
## Tech Stack

- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)

## Project Structure

```text
/
├── astro.config.mjs
├── package.json
├── public/
├── src/
│   ├── assets/
│   │   └── posters/
│   ├── components/
│   ├── data/
│   ├── layouts/
│   ├── pages/
│   └── styles/
└── README.md
```

## Development

```sh
npm install
npm run dev
```

## Build and Preview

```sh
npm run build
npm run preview
```
